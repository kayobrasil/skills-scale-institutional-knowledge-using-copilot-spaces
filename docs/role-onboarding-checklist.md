# Role Onboarding Checklist (for new project members)

This checklist helps new joiners understand who to contact and how to get productive quickly.

- [ ] Read project One-pager and Project README
- [ ] Read docs/octoacme-roles-and-personas.md
- [ ] Identify your primary contact(s) for product, delivery, and engineering
- [ ] Confirm access to required systems (repo, CI, dashboards)
- [ ] Pair with a team member for codebase walkthrough
- [ ] Review current sprint board and assigned tasks
- [ ] Review release readiness and deployment runbooks
- [ ] Add yourself to relevant communication channels and incident on-call rotations (if applicable)
